Accessibility Statement

Our Commitment

ImpactMojo is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to make our educational content available to all learners.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 level AA standards. Our current conformance status is:

Partially Conformant

This means that some parts of the content do not fully conform to the accessibility standard, but we are actively working to achieve full compliance.

Accessibility Features

Keyboard Navigation

Our website can be navigated using a keyboard. Key features include:

  • Tab navigation through all interactive elements
  • Enter key to activate buttons and links
  • Escape key to close modals and menus
  • Arrow keys for navigation within components

Visual Design

  • High contrast ratios meeting WCAG AA standards
  • Resizable text without loss of functionality
  • Clear focus indicators on interactive elements
  • Dark mode option to reduce eye strain
  • No reliance on color alone to convey information

Screen Reader Support

  • Semantic HTML structure
  • ARIA labels and descriptions where needed
  • Proper heading hierarchy
  • Alternative text for images
  • Form labels and error messages

Responsive Design

  • Mobile-friendly interface
  • Touch-friendly interactive elements
  • Readable text on all screen sizes
  • Simplified navigation on mobile devices

Keyboard Shortcuts

Action Shortcut
Search courses Ctrl + K or Cmd + K
Toggle dark mode Ctrl + D or Cmd + D
Close modal/dialog Esc
Navigate between sections Tab / Shift + Tab

Known Limitations

While we strive for accessibility, some areas are still being improved:

  • Some older PDF documents may not be fully accessible
  • Complex data visualizations may need additional text descriptions
  • Video content is being updated with captions and transcripts
  • Some third-party embedded content may have limited accessibility

Browser Compatibility

ImpactMojo is designed to work with modern browsers that support current web standards:

  • Chrome (version 90+)
  • Firefox (version 88+)
  • Safari (version 14+)
  • Edge (version 90+)

Assistive Technologies

We test our website with various assistive technologies including:

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (macOS/iOS)
  • TalkBack (Android)

Feedback and Contact

We Welcome Your Feedback

If you encounter any accessibility barriers or have suggestions for improvement, please contact us:

  • Email: accessibility@impactmojo.in
  • Response time: Within 48 hours

Please provide the following information:

  • Description of the accessibility issue
  • Web page URL where you encountered the issue
  • Your assistive technology (if applicable)
  • Your browser and operating system

Technical Specifications

Accessibility of ImpactMojo relies on the following technologies:

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • WAI-ARIA 1.1

Assessment and Remediation

We regularly assess our website's accessibility through:

  • Automated testing tools (axe DevTools, WAVE)
  • Manual keyboard navigation testing
  • Screen reader testing
  • User feedback and testing

Legal and Standards Compliance

We aim to comply with:

  • Web Content Accessibility Guidelines (WCAG) 2.1 Level AA
  • Rights of Persons with Disabilities Act, 2016 (India)
  • Section 508 of the Rehabilitation Act (where applicable)
